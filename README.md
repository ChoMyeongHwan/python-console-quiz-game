# 🎮 Python 퀴즈 게임

## 프로젝트 개요
터미널에서 동작하는 Python 기초 퀴즈 게임입니다.<br>
퀴즈 풀기, 추가, 목록 확인, 점수 관리 기능을 제공하며 JSON 파일로 데이터를 저장해 프로그램 재실행 후에도 유지됩니다.

## 퀴즈 주제 선정 이유
Python 기초 문법을 주제로 선정했습니다.<br>
학습한 내용을 직접 퀴즈로 만들며 복습 효과를 높이기 위해 선택했습니다.

## 실행 방법
```bash
python main.py
```

## 기능 목록
| 기능 | 설명 |
|------|------|
| 퀴즈 풀기 | 저장된 퀴즈를 순서대로 풀고 결과 확인 |
| 퀴즈 추가 | 새 퀴즈(문제/선택지4개/정답)를 등록 |
| 퀴즈 목록 | 등록된 퀴즈 전체 목록 출력 |
| 점수 확인 | 최고 점수 및 정답률 확인 |
| 종료 | 데이터 저장 후 프로그램 종료 |

## 파일 구조
```
프로젝트/
├── main.py           # 프로그램 진입점
├── quiz.py           # Quiz 클래스
├── quiz_game.py      # QuizGame 클래스
├── state.json        # 데이터 저장 파일
└── README.md
```

## 데이터 파일 설명 (state.json)
- 경로: 프로젝트 루트 / state.json
- 인코딩: UTF-8
- 역할: 퀴즈 목록과 최고 점수를 영구 저장

```json
{
  "quizzes": [
    {
      "question": "문제 내용",
      "choices": ["선택1", "선택2", "선택3", "선택4"],
      "answer": 1
    }
  ],
  "best_score": 3
}
```
